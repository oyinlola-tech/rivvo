# Security Policy

Security is taken seriously for this project. Please report vulnerabilities responsibly and do not publicly disclose security issues without coordination.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it directly to the project owner:

**Owner:** Oluwayemi Oyinlola Michael  
**Portfolio:** `oyinlola.site`  
**Tech Firm:** `telente.site`

When reporting, include:

- A clear description of the issue.
- Steps to reproduce.
- Expected vs actual behavior.
- Affected components and versions (if known).
- Any proof‑of‑concept or screenshots (optional).

## Disclosure Process

1. **Initial triage** within 3–7 days.
2. **Assessment and fix** with internal tracking.
3. **Coordinated disclosure** once a fix is available.

We appreciate responsible disclosure and will credit reporters upon request.

## Scope

This policy covers all code and configurations within this repository, including the backend API, frontend UI, and any deployment configurations.
